🏠 CD-Startpage Manual

Unified service portal dashboard

Overview

Central landing page with links to all SOC services. Displays service status, quick access, documentation.

Features

  • Service Links — Quick launch to SOCint, SOCops, SBOMguard, honeypot, Wazuh dashboard
  • Status Indicators — Green/red status for each service
  • Documentation — Links to user manuals
  • Health Check — Real-time service availability

Service Tiles

  • SOCint (Threat Intelligence) — Port 3000 (frontend) / 8000 (API)
  • SOCops (Wazuh Workbench) — Port 8081
  • SBOMguard (SBOM Tracker) — Port 8082
  • NetScaler Honeypot — Port 8083
  • Wazuh Dashboard — Port 8080

Accessing

http://localhost      # Standalone Startpage port 8080
http://localhost      # Bundle deployment port 80

Configuration

Edit .env to customize:

  • Service URLs (if not on localhost)
  • Title, logo, theme
  • Health check endpoints

Health Checks

Startpage polls service health endpoints and displays status:

  • http://localhost:8000/api/health — SOCint API
  • http://localhost:8081/api/stats — SOCops
  • http://localhost:8082/api/health — SBOMguard
  • http://localhost:8083/health — Honeypot

Customization

Modify app.py to:

  • Change colors, fonts, layout
  • Add/remove service tiles
  • Customize health check logic

Deployment

# Standalone
python3 app.py    # Runs on port 8080

# Docker Compose (bundled)
docker compose up startpage  # Runs on port 80
